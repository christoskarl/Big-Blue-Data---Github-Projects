# Project README

## Overview
This project explores an Amazon product dataset collected in 2022. It analyzes relationships between product ratings, discounts, and prices across **1,348 products** spanning multiple categories, with the largest being **Electronics**, **Computers & Accessories**, and **Home & Kitchen**.

The goal is to clean and analyze the dataset to uncover trends, correlations, and insights related to pricing strategies and consumer behavior.

---

## Project Structure
The repository is organized into the following folders:

### **1. `data/`**
Contains the dataset used for this project. The dataset includes:
- 1,348 total products
- Product category information
- Product ratings
- Pricing and discount details

### **2. `cleaning/`**
Includes the code used for cleaning and preprocessing the dataset. Typical steps may include:
- Handling missing values
- Standardizing category names
- Converting column types
- Removing duplicates

### **3. `analysis/`**
Contains the analysis code used to examine relationships between:
- Ratings and prices
- Discounts and final prices
- Category-level pricing trends

This folder may include statistical summaries, visualizations, and correlation computations.

### **4. `presentation/`**
Includes materials used to present project findings. This may include:
- Slide decks
- Visualizations
- Summary reports

### **5. `README.md`**
This file. Provides an overview of the project, dataset, and repository structure.
